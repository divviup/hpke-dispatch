Struct TurboShake 

Source 
pub struct TurboShake<const RATE: usize, const DS: u8> { /* private fields */ }
Expand description

TurboSHAKE hasher generic over rate and domain separator.

Domain separator DS MUST be in the range 0x01..=0x7f. Use DEFAULT_DS if you want the default value.

Rate MUST be either 168 or 136 for TurboSHAKE128 and TurboSHAKE256 respectively.
